# Python and Visual Studio Code

This layer installs Python, Visual Studio Code, and a small set of Python-focused extensions for VS Code.

Optionally, the layer prepares `pip` to use the session host proxy so students can install Python packages during the exam. If allowed, make sure the proxy whitelist in `properties.json5` allows the required Python package hosts.

Also, it contains a commented out example to prepare VS code for use Jupyter notebooks.

A previous version of this layer has been tested extensively in real exam settings.
