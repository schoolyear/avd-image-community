# Experimental Windows 11 start menu improvement

This layer ensures the start menu is responsive when clicked for the first time by a student.

The start menu in Windows 11 is slow to open (~10 seconds), when clicked for the first time on a new profile. This layer remedies this
by opening and closing the start menu once automatically.

Note: this layer adds ~10 seconds to the 'Waiting for your exam environment to start' phase.
